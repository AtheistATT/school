=======================================================
Тема 2.6. Финансы и банки: Как деньги работают в экономике
=======================================================

Основные понятия
================

Давайте представим мир финансов как большую транспортную систему для денег.

.. list-table:: Кто есть кто в мире денег?
   :widths: 20 35 45
   :header-rows: 1

   * - Термин
     - Простое объяснение
     - Пример из жизни
   * - **Финансы**
     - Не просто деньги, а их **движение** и все отношения вокруг них
     - Зарплата, покупка квартиры, оплата учебы, налоги
   * - **Финансовая система**
     - "Кровеносная система" экономики
     - Все законы, организации и потоки денег в стране
   * - **Финансовый институт**
     - **Посредник** между теми, у кого есть деньги, и теми, кому они нужны
     - Банк, страховая компания, пенсионный фонд
   * - **Финансовые инструменты**
     - "Документы-обязательства", подтверждающие финансовые права
     - Акция, облигация, страховой полис, валюта

💡 **Главная идея:** Финансовые институты — это **мосты**, соединяющие сберегателей и заемщиков.

Зачем нужны финансы государству?
=================================

Финансы выполняют 4 главные функции в экономике:

1. **Распределительная** — Собрать деньги (налоги) и распределить их на общие нужды

   .. code-block:: none

      Налоги граждан и компаний → Бюджет государства →
      │
      ├──→ Школы и детсады
      ├──→ Больницы и поликлиники
      ├──→ Дороги и мосты
      ├──→ Армия и полиция
      └──→ Наука и культура

2. **Контрольная** — Следить, чтобы деньги тратились по назначению
3. **Регулирующая** — Управлять экономикой через налоги и госрасходы
4. **Стабилизирующая** — Обеспечивать стабильность (пенсии, пособия)

.. note:: **Пример:** Когда государство строит новую школу — это **распределительная** функция. Когда проверяет, как потрачены деньги на ремонт — это **контрольная** функция.

Банковская система: двухэтажная конструкция
============================================

Банковская система России похожа на двухэтажное здание:

.. code-block:: none
   :class: center

   ┌─────────────────────────────────────────────┐
   │           ЦЕНТРАЛЬНЫЙ БАНК (ЦБ РФ)          │
   │  "Директор" всей банковской системы         │
   │  • Печатает деньги                          │
   │  • Устанавливает правила                    │
   │  • Контролирует банки                       │
   └─────────────────────────────────────────────┘
                         │
                         │ устанавливает правила
                         ▼
   ┌─────────────────────────────────────────────┐
   │         КОММЕРЧЕСКИЕ БАНКИ                  │
   │  "Финансовые магазины" для всех             │
   │  • Принимают вклады                          │
   │  • Выдают кредиты                           │
   │  • Делают переводы                          │
   └─────────────────────────────────────────────┘
            │                    │
            │ вклады             │ кредиты
            ▼                    ▼
     [Население и компании]  [Население и компании]

Уровень 1: Центральный банк (ЦБ РФ)
------------------------------------

**Роль:** Главный банк страны, "банк для банков"

**Основные функции:**

✅ **Единственный эмитент** — только он печатает наличные рубли
✅ **Банковский регулятор** — выдает лицензии и контролирует банки
✅ **Кредитор последней инстанции** — дает кредиты банкам в кризис
✅ **Хранитель золотовалютных резервов**

.. important::

   ЦБ **не работает** с обычными людьми и компаниями! Он устанавливает правила для всех банков.

Уровень 2: Коммерческие банки
------------------------------

**Роль:** Финансовые магазины для населения и бизнеса

**Основная функция:**
   **Привлекают вклады → Выдают кредиты**

**Виды коммерческих банков:**

* **Универсальные** — работают со всеми (Сбер, ВТБ, Тинькофф)
* **Специализированные:**

  *Сберегательные* — работают с вкладами населения
  *Ипотечные* — выдают кредиты под залог недвижимости
  *Инвестиционные* — финансируют бизнес-проекты

Как банк зарабатывает деньги?
==============================

Банк работает по простой схеме:

.. code-block:: none

   ┌─────────────────────────────────────────────────────┐
   │              ДОХОД БАНКА = 10%                      │
   │  (разница между 15% и 5%)                           │
   └─────────────────────────────────────────────────────┘
            ▲                            ▲
            │                            │
   ┌────────┴────────┐        ┌──────────┴──────────┐
   │ Кредит заемщику │        │ Вклад от населения  │
   │ под 15% годовых │        │ под 5% годовых      │
   └─────────────────┘        └─────────────────────┘
            │                            │
            ▼                            ▲
   ┌─────────────────┐        ┌─────────────────────┐
   │   Заемщики:     │        │   Вкладчики:        │
   │ • Предприятия   │        │ • Население         │
   │ • Люди на       │        │ • Компании          │
   │   крупные       │        │                     │
   │   покупки       │        └─────────────────────┘
   └─────────────────┘

Типы банковских операций:

.. list-table::
   :widths: 25 35 40
   :header-rows: 1

   * - Тип операции
     - Что это?
     - Пример
   * - **Пассивные**
     - **Привлечение** денег в банк
     - Вклады, выпуск акций банка
   * - **Активные**
     - **Размещение** денег для дохода
     - Кредиты, инвестиции
   * - **Комиссионные**
     - **Услуги** за плату
     - Переводы, аренда сейфов

💸 **Формула дохода банка:**

   **Доход = (Процент по кредитам - Процент по вкладам) + Комиссии**

Что такое кредит и как он работает?
====================================

**Кредит** — это деньги в долг на **строгих условиях**.

4 главных принципа кредитования:

.. code-block:: none

   ┌─────────────┬──────────────┬─────────────┬─────────────────┐
   │Возвратность │ Срочность    │ Платность   │ Обеспеченность  │
   ├─────────────┼──────────────┼─────────────┼─────────────────┤
   │Обязательно  │Вернуть       │Заплатить    │Залог или        │
   │вернуть      │в указанный   │проценты     │поручитель       │
   │             │срок          │             │                 │
   └─────────────┴──────────────┴─────────────┴─────────────────┘

Виды кредитов в жизни:
-----------------------

.. list-table:: Основные виды кредитов
   :widths: 25 25 25 25
   :header-rows: 1

   * - Вид кредита
     - На что дают
     - Срок
     - Особенности
   * - **Потребительский**
     - Телефон, отпуск, ремонт
     - 1-5 лет
     - Высокий процент, быстрое оформление
   * - **Ипотечный**
     - Покупка жилья
     - 10-30 лет
     - Залог — сама квартира, низкий процент
   * - **Банковский (для бизнеса)**
     - Развитие компании
     - 1-10 лет
     - Нужен бизнес-план, проверка компании
   * - **Государственный**
     - Поддержка фермеров, студентов
     - Разный
     - Льготные условия, целевое использование

Круговорот денег в экономике
=============================

.. code-block:: none

      [ВКЛАДЧИКИ]
      (у них есть свободные деньги)
           │
           ▼ вкладывают под 5%
      ┌─────────────┐
      │   БАНК      │←─── получает доход 10%
      └─────────────┘      (15% - 5%)
           │
           ▼ выдает кредиты под 15%
      [ЗАЕМЩИКИ]
      (им нужны деньги на развитие)
           │
           ▼ вкладывают в бизнес
      ┌─────────────┐
      │  ЭКОНОМИКА  │ ←── растет!
      └─────────────┘
           │
           ▼ получают прибыль
      [ЗАЕМЩИКИ]
           │
           ▼ возвращают кредит с %
      ┌─────────────┐
      │   БАНК      │
      └─────────────┘
           │
           ▼ выплачивает % вкладчикам
      [ВКЛАДЧИКИ]
           │
           └──→ круг замыкается!
