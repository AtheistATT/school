Тема 2.9. Рынок труда и безработица
====================================

Что такое рынок труда?
------------------------
Это особый **"рынок"**, где встречаются:
*   **Те, кто ищет работу** (предлагают свои силы и умения) — **НАЁМНЫЕ РАБОТНИКИ**.
*   **Те, кому нужны работники** (готовы платить за труд) — **ПРЕДПРИЯТИЯ** (работодатели).

**Главный товар на этом рынке — НЕ человек, а его ТРУДОВЫЕ УСЛУГИ** (рабочее время, знания, навыки).

Кто ещё участвует?
*   **ПРОФСОЮЗЫ** — защищают права работников.
*   **ГОСУДАРСТВО** — устанавливает правила (например, минимальную зарплату).

Что определяет спрос и предложение на рынке труда?
-----------------------------------------------------
+----------------------+--------------------------------------------------------+
| **Спрос на труд**    | **Предложение труда** (кто и сколько хочет работать)   |
| (кого и сколько      |                                                        |
| ищут работодатели)   |                                                        |
+======================+========================================================+
| • Успешность фирмы.  | • Численность населения.                               |
| • Уровень технологий.| • Уровень образования и квалификации людей.            |
| • Спрос на товары    | • Размер предлагаемой зарплаты.                        |
|   фирмы.             | • Государственная политика.                            |
+----------------------+--------------------------------------------------------+

В результате "торга" устанавливается **равновесная цена** — средний уровень **ЗАРАБОТНОЙ ПЛАТЫ** в профессии.

Особенности рынка труда
------------------------
1.  **Важен "человеческий капитал"** — это запас знаний, здоровья, опыта работника. Чем он больше, тем выше может быть зарплата.
2.  **Оплата — это не только "голая" зарплата**, но и бонусы, льготы, обучение, комфортные условия.
3.  **Работники и работодатели заинтересованы в долгом сотрудничестве** — это выгодно обеим сторонам.

Заработная плата
-----------------
**Зарплата** — материальное вознаграждение за труд.

*   **Номинальная зарплата** — сумма денег в вашей зарплатной ведомости.
*   **Реальная зарплата** — то, что вы можете купить на эти деньги с учётом цен. **Именно она важна!**

**Нижняя граница** — **ПРОЖИТОЧНЫЙ МИНИМУМ** (сумма на самое необходимое для жизни).

Виды и формы зарплаты
----------------------

+--------------------+----------------------------------------------------------------+
| **Виды зарплаты**  | **Формы оплаты**                                               |
+====================+================================================================+
| **Повременная**    | Оплата за отработанное время (час, день, месяц).               |
|                    | *Пример: оклад, почасовая ставка.*                             |
+--------------------+----------------------------------------------------------------+
| **Сдельная**       | Оплата за количество и качество сделанной работы.              |
|                    | *Пример: оплата за каждую изготовленную деталь.*               |
+--------------------+----------------------------------------------------------------+

Что такое безработица?
-----------------------
Это ситуация, когда часть **экономически активного населения** (тех, кто хочет и может работать) **не может найти работу**.

Кого считают безработным?
1.  Не имеет работы.
2.  Ищет её (обратился в службу занятости).
3.  Готов приступить к работе.

Виды безработицы (главные)
----------------------------

+----------------------+--------------------------------------------------------------+
| **Вид**              | **Суть и пример**                                            |
+======================+==============================================================+
| **Фрикционная**      | ВРЕМЕННАЯ безработица "между работами".                      |
| (от 1 до 3 месяцев)  | *Пример: Выпускник ищет первую работу; человек уволился,     |
|                      | чтобы найти лучшее место.*                                   |
+----------------------+--------------------------------------------------------------+
| **Структурная**      | Безработица из-за НЕСООТВЕТСТВИЯ спроса и предложения.       |
|                      | *Пример: Исчезла профессия (например, телефонистка), а       |
|                      | появилась новая (IT-специалист). Нужно переучиваться.*       |
+----------------------+--------------------------------------------------------------+
| **Циклическая**      | Самая тяжёлая. Возникает из-за СПАДА в экономике всей        |
| (или массовая)       | страны. Рабочих мест не хватает вообще.                      |
|                      | *Пример: Кризис 2008-2009 гг., закрытие многих предприятий.* |
+----------------------+--------------------------------------------------------------+

**Естественный уровень безработицы** = фрикционная + структурная. Это **нормальное** явление в экономике (обычно 5-7%).

Последствия безработицы
------------------------

+------------------------+--------------------------------------------------------+
| **Негативные**         | **Позитивные (для экономики в целом)**                 |
+========================+========================================================+
| • Падение уровня жизни | • Формируется "резерв" рабочей силы для роста.         |
|   и потребительского   |                                                        |
|   спроса.              | • Сдерживается инфляция (профсоюзы не требуют резко    |
| • Потеря квалификации. |   поднимать зарплату).                                 |
| • Моральная травма,    | • Повышается трудовая дисциплина у тех, кто работает   |
|   рост преступности.   |   (ценят своё место).                                  |
| • Недовыпуск продукции.|                                                        |
+------------------------+--------------------------------------------------------+

Что делает государство?
------------------------
Борясь с безработицей, государство:
1.  **Выплачивает пособия** по безработице.
2.  **Организует переобучение** (борьба со структурной безработицей).
3.  **Создаёт новые рабочие места** (например, через госзаказы, поддержку малого бизнеса).
4.  **Устанавливает минимальную зарплату**.
5.  **Развивает экономику**, чтобы не допустить циклического спада.

**Вывод:** Рынок труда — важнейший элемент экономики. Безработица — его неизбежный спутник, но государство и общество должны смягчать её негативные последствия.
